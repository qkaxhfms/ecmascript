Chrome Debugging

디버깅(debugging)은 스크립트 내 에러를 검출해 제거하는 일련의 과정을 의미합니다.
모던 브라우저와 호스트 환경 대부분은 개발자 도구 안에 UI 형태로 디버깅 툴을 구비해 놓습니다.
디버깅 툴을 사용하면 디버깅이 훨씬 쉬워지고,
실행 단계마다 어떤 일이 일어나는지를 코드 단위로 추적할 수 있습니다.

  1. 중단점을 만났을 때
  2. debugger문 만났을 때
  3. 에러가 발생했을 때(개발자 도구가 열려있고 버튼이 '활성화’되어있는 경우
스크립트 실행이 중지되면 중단 시점을 기준으로 변수에 어떤 값이 들어가 있는지 확인할 수 있습니다.
